## Description

Rebuild an existing WP Engine Headless Applications

## Usage

```bash
$ wpe apps rebuild $APP_NAME [OPTIONS]
```

## Extended Description

The `wpe apps rebuild` command will rebuild a specified WP Engine Headless Application. Use the [`wpe apps list`](./list) command to get a list of the apps you can rebuild.

## Options

```
--help, -h    Help for rebuild
```

## Examples

### Rebuild "myapp"

```bash
$ wpe apps rebuild myapp
```
