The `wpe context` command group allows you to get and set account context for the WP Headless CLI.

## Commands

- [**wpe context get**](./context/get): Get the account context for the WP Engine Headless CLI
- [**wpe context set**](./context/set): Set the account context for the WP Engine Headless CLI
