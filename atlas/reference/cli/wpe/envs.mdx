The `wpe envs` command group allows you to manage environments in an application using the WP Headless CLI.

## Commands

- [**wpe envs delete**](./envs/delete): Remove a deployed environment from an application
- [**wpe envs purge**](./envs/purge): Manually purge the domain cache for a specified application environment
- [**wpe envs rebuild**](./envs/rebuild): Manually invoke a rebuild of a specified application environment
