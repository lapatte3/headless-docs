**Note**: These tutorials assume that you have already [configured the plugin with your BigCommerce API Account credentials](connecting-bigcommerce.mdx) and [performed a product sync](performing-sync.mdx).

Atlas Commerce Blocks lets you place product content synced from BigCommerce in your WordPress content using the WordPress editor data from BigCommerce into WordPress. Using the familiar blocks editorr, you can build pages and posts with product content.

## Install the Atlas Commerce Connector plugin

Atlas Commerce Blocks comes pre-installed with the Atlas Commerce Blueprint option via our User Portal and via WP Local.

![Blueprint Plugins](../images/blueprint-plugins.png)

- Check out the demo of [Atlas Commerce Blueprint](#)
- Try out a [sandbox version of Atlas](#) where you can install Atlas Commerce Blueprint

## Available Blocks

Choose from a variety of blocks by accessing the block editor and searching **Atlas**. This will bring up the available blocks that are connected to your BigCommerce products. Build containers, accordions, product blocks with various configurations and more.

![Atlas Commerce Blocks](../images/atlas-commerce-blocks.png)

### Troubleshooting

---

1. If there are no products showing when using the product blocks, please ensure you have performed an up to date sync.
