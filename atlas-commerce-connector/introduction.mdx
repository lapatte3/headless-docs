Atlas Commerce Connector for BigCommerce lets you sync product data from BigCommerce into WordPress. Using an intuitive interface, you can connect BigCommerce API Accounts to WordPress, sync your product data and use them as blocks when working in the WordPres editor or custom post types in WPGraphQL when working with headless.

## Install the Atlas Commerce Connector plugin

Atlas Commerce Connector for BigCommerce comes pre-installed with the Atlas Commerce Blueprint option via our User Portal and via WP Local.
